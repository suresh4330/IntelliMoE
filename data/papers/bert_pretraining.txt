Title: BERT: Pre-training of Deep Bidirectional Transformers for Language Understanding
Authors: Devlin, Chang, Lee, Toutanova (Google AI Language, 2019)
Published: NAACL 2019 (Best Paper Award)

ABSTRACT
We introduce a new language representation model called BERT, which stands for Bidirectional Encoder Representations from Transformers. Unlike recent language representation models, BERT is designed to pre-train deep bidirectional representations from unlabeled text by jointly conditioning on both left and right context in all layers. As a result, the pre-trained BERT model can be fine-tuned with just one additional output layer to create state-of-the-art models for a wide range of tasks.

MOTIVATION AND BACKGROUND
Two existing strategies for applying pre-trained language representations to downstream tasks:
1. Feature-based (e.g., ELMo): Uses pre-trained representations as additional features
2. Fine-tuning (e.g., GPT): Introduces minimal task-specific parameters and fine-tunes all parameters

GPT (unidirectional) processes text left-to-right only, limiting the model's ability to understand context from both directions. ELMo concatenates left-to-right and right-to-left LSTMs but does not achieve deep bidirectional fusion.

BERT ARCHITECTURE
BERT uses the Transformer encoder stack (from "Attention Is All You Need"):
- BERT-base: 12 layers, 768 hidden dim, 12 attention heads, 110M parameters
- BERT-large: 24 layers, 1024 hidden dim, 16 attention heads, 340M parameters

Input representation combines three embeddings:
1. Token embeddings (WordPiece tokenizer, 30,000 vocab)
2. Segment embeddings ([A] vs [B] sentence markers)
3. Position embeddings (learned, not sinusoidal like original Transformer)

PRE-TRAINING TASKS
Task 1 — Masked Language Modeling (MLM):
  - 15% of input tokens are randomly masked
  - Of these: 80% replaced with [MASK], 10% random token, 10% unchanged
  - Model predicts original token at masked positions
  - Enables truly bidirectional pre-training

Task 2 — Next Sentence Prediction (NSP):
  - Model receives two sentences A and B
  - 50% chance B is the actual next sentence (IsNext), 50% random (NotNext)
  - Model predicts [IsNext] / [NotNext] via [CLS] token
  - Captures inter-sentence relationships important for QA and NLI

FINE-TUNING PROCEDURE
BERT can be fine-tuned for virtually any NLP task by adding a task-specific output layer:
- Classification: linear layer on [CLS] token
- Token labeling (NER): linear layer on each token representation
- Question answering: two linear layers predicting start/end span positions
- Only 1-4 epochs of fine-tuning typically needed on downstream tasks

RESULTS
BERT achieved state-of-the-art on 11 NLP benchmarks:
- GLUE: 80.5% (7.7% improvement over previous SOTA)
- SQuAD v1.1: 93.2 F1 (human: 91.2)
- SQuAD v2.0: 83.1 F1
- SWAG: 86.3% accuracy
- MultiNLI: 86.7% accuracy

IMPACT AND LEGACY
BERT fundamentally changed NLP and influenced nearly all subsequent language models:
- RoBERTa: Removed NSP, trained longer with more data
- DistilBERT: 40% smaller, 60% faster, 97% of BERT performance
- ALBERT: Parameter sharing to reduce model size
- SciBERT, BioBERT, LegalBERT: Domain-specific BERT variants
- Established masked language modeling as a core pre-training paradigm
